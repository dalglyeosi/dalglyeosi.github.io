주흔작 계산기
사용방법 및 유의사항
- 방어구, 무기는 적혀진대로 입력해주세요.
  (데벤져는 설명란 하단을 참고해주세요.)
  (무기의 스타포스 정보는 필요하지 않습니다.)
- 증가값이 없을 경우 0을 입력 해주세요.
- 위치 참고는 [여기]를 클릭해주세요.
- 렙제는 글러브에만 선택해주시면 됩니다.
- 일반 무기/방어구는 120제 이상의 아이템에만 적용됩니다.
- 글러브는 주스텟에는 공/마,
  부스텟에는 강화된 스타포스 수를 써주세요
  (아이템에 'n성 강화 적용'이라고 적혀있는데 여기서 n을 입력)
- 전사ㅣ주스텟 STR / 부스텟 DEX
  궁수ㅣ주스텟 DEX / 부스텟 STR
  도적ㅣ주스텟 LUK / 부스텟 DEX
  법사ㅣ주스텟 INT / 부스텟 LUK
  해적ㅣ주스텟 STR 또는 DEX / 부스텟 DEX 또는 STR
- 데벤져 무기일 경우 주스텟에 HP, 부스텟에 MP를 써주세요.
- 데벤져 방어구일 경우 주스텟에는 HP,
  부스텟에는 강화된 스타포스 수를 써주세요.
- 데벤져 방어구일 경우 150제 아이템에만 적용됩니다.
